key(Key, Options) -> optic:optic()

  • Key = term()
  • Options = optic:variations()

Key: The key to focus on.
Options: Common optic options.

returns: An opaque optic record.

Focus on the value of a property list key. As keys may be duplicated, this may be multiple values. If the value is not given, it defaults to the atom true.

Example:

  > optic:get([optic_proplists:key(first)], [{first, 1}, {second, 2}]).
  {ok,[1]}

property/2

property(Key, Options) -> optic:optic()

  • Key = term()
  • Options = optic:variations()

Key: The key to focus on.
Options: Common optic options.

returns: An opaque optic record.

Focus on a property in a property list by key. As keys may be duplicated, this may be multiple properties. If the value is not given, it defaults to the atom true. If the key is modified, the optic is no longer well behaved.

Example:

  > optic:get([optic_proplists:property(first)], [{first, 1}, {second, 2}]).
  {ok,[{first,1}]}
