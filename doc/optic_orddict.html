Description

A set of optics specific to orddicts.

As orddicts are internally represented as a list of pairs, the type checks used here are not as reliable as those used for other optics. Please ensure via other means that these optics are only used with actual orddicts.

Key: The key to focus on.
Options: Common optic options.

returns: An opaque optic record.

Focus on the association for an orddict key. An association is the tuple of a orddict key and value. If the key is modified, the optic is no longer well behaved.

Example:

  > optic:get([optic_orddict:association(first)],
              orddict:from_list([{first, 1}, {second, 2}])).
  {ok,[{first,1}]}
